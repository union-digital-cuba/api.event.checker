# API Event Checker 💻
## Description 📚
 API para controlar la subscripcion a eventos, asi como gestionar los mensajes y notificaciones que son necesarias para el funcionamiento de las otras APIs 
## Installation 💡 
 npm install
npm run dev 
## App features 🪐
 - API
- WebSocket
- Agendar
